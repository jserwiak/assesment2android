User Manual

1. Introduction

This user manual has been prepared to help new users of the application and

make navigation smoothly and to create the best experience possible.

2. Welcome Page

Welcome Page is the starting point of the application.

There user can choose the color theme by clicking on:

Dark button – dark theme

Light button – light theme

Selected theme will be saved and next time when user opens the application, it will be applied

 

There is also a “API call” button that calls WEB API  and displays data as toast.

 

To navigate to the Contact list page simply swipe left on the screen

3. Contact List Page

 

In the contact list page user can see all the contacts, that are saved in the database.

There are multiple functionalities, that will help the user to navigate through the contacts,

 search and sort the contacts. User can also display contact details, add new contact,

edit a contact and also delete.

 

 

 

3.1 Search contacts

To search for a contact user can either use a left-hand side panel with letters or search bar.

Side panel

If user want to navigate to contact, that starts with letter he needs to click on the letter button in

the side panel and list will scroll to the position of the corresponding contact.

Search bar

User can access a search bar by simply clicking the button shown below.

After clicking the button a search bar will appear.

Here user can type the contact he are looking for and press search button

       

3.2 Sorting contacts

As default contacts will be displayed in ascending order. To change sorting user can click

on one of the below buttons.

 

3.3 Displaying contact details.

If user wants to display details of a contact, he can chose the contact to display by clicking

 on it on the contact list. Contact details will be shown in the Detail Page. To navigate back t

o the list of contacts, user can press Back button

 

3.4 Adding contact.

To add a new contact user need to press a “Plus” button as shown on the picture below. Add contact page

will be displayed, where user will be asked to fill the details. After doing that and pressing “Add” button,

contact will be saved.

 

3.5 Editing existing contact

To edit a contact user need to swipe the contact left on the contact list to reveal the edit and

delete button. After clicking on the edit button, Edit page will be displayed. After making changes

and pressing “Update” button, changes will be saved and user will be taken back to the contact list

 page.

3.6 Deleting the contact.

To delete the contact user need to swipe left on the contact to reveal delete button,

and click on the “Bin” button. He will be ask for confirmation to avoid accidental deletion.

If user confirms to delete the contact by pressing delete, contact will be removed, and user

will be taken back to contact list page.
